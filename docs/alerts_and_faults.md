# Alerts and Fault Handling

## Buzzer Alerts
- Over-temperature condition
- Sensor disconnection or failure
- System startup error

## LED Indicators
- Alarm LED flashes during faults
- Wi-Fi LED shows connection status

## Wi-Fi Alerts
- Temperature out of range
- Humidity out of range
- System fault notifications

## Fault Handling
- System continues in safe mode
- Heater disabled during critical faults
- Alerts remain active until fault clears
